02
신규 서비스 확장

일상에서 미래까지, 우리의 서비스 영역을 넓힙니다.

퀵대장(화물) 런칭

오토바이 중심에서 화물 운송까지 확장하여 종합 퀵커머스 플랫폼으로 도약하여 신규 수익원을 창출합니다.

  • →
    서비스 분류 체계
    퀵: 오토바이 (초소형~중형) / 화물: 다마스, 라보, 1톤 (대형)
  • →
    차등 요금 정책
    상하차 방법별 추가 요금 (고객직접 0원 ~ 기사담당 최대 16,000원)
  • →
    8단계 통합 주문 플로우
    상품 크기 → 차량 선택 → 상하차 옵션 → 주소 → 시간 → 확인 → 등록
  • →
    스마트 UI/UX 설계
    상품 크기별 자동 차량 필터링, 아코디언 형식 통합 주문으로 편의성 극대화

더 스마트한 관리 기능

    고객사 맞춤 공지
    고객사가 직접 관리하는 공지사항 시스템으로 이미지, 팝업 등 다양한 형태의 맞춤형 공지 제공

    사장님 결제 취소

    퀵대장 부서 예산 관리

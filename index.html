    제휴식당 탐색 개선

    ✓ 제휴점 정보의 탐색 편의성 개선
    ✓ 위치 기반 필터링으로 조회 성능 향상 및 서버 부하 최적화
  • 🤝
    함께결제 UI/UX 사용성 개선

    함께결제 결제 과정을 직관적이고 편리하게 처리할 수 있도록 사용자 인터페이스를 개선

    ✓ 금액, 메뉴, 인원 변경시 직관적 UI 제공
    ✓ 캡틴코드 결제도 함께결제 방식으로 이용

구내식당 & 원격주문 서비스 확장

대형 고객사의 구내식당 운영 효율화와 디지털 전환 요구에 대응하여 통합 솔루션을 제공하고 고객사 만족도를 높입니다.

  • ⚡
    원격주문 (스마트오더)

    미리 주문하고 바로 픽업하는 간편한 시스템으로 사내 카페나 또는 구내식당의 사용에 대해서 사무실에서 확인하고 이용가능한 서비스를 제공

    ✓ 원격 주문 및 결제 시스템
    ✓ 고객사 자체 카페 운영 효율성 및 편의성 지원
    ✓ 구내식당 간식, 간편식 예약 수령 서비스 제공
  • 📋
    식단표
    대형 고객사의 구내식당 디지털화 요구에 대응하여 실시간 식단 정보 제공으로 고객사 만족도 증대
  • 🖥️
    식권발급기 키오스크 연동
    대형 고객사 신규 영업 지원을 위한 통합 솔루션 제공으로 고객사 락인 효과 확대 및 장기 파트너십 강화

🚚

종합 총무 서비스 플랫폼으로의 성장

식대, 복지 관리에서 퀵 화물 운송까지, 종합 총무 서비스 플랫폼으로 한 단계 성장합니다.

  • 🎯
    명확한 서비스 구분
    🏍️ 퀵서비스: 오토바이로 작은 물건 / 🚚 화물서비스: 다마스, 라보, 1톤으로 큰 물건
  • 💰
    상하차 서비스별 요금제
    고객이 직접 하면 0원, 기사님이 도와주면 추가 요금 (최대 16,000원)
  • 📱
    스마트한 주문 시스템
    물건 크기만 선택하면 적합한 차량이 자동으로 필터링되고, 복잡한 옵션들이 단계별로 쉽게 안내되는 똑똑한 시스템
  • ✨
    직관적인 사용자 경험
    복잡한 설정 없이 아코디언 방식으로 단계별 선택이 쉬워요

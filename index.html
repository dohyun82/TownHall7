02
새로운 영역으로의 도전

우리의 가능성을 더 넓은 영역으로 확장합니다

🚚

퀵대장 화물 서비스 출시

기존 오토바이 퀵에서 화물 운송까지, 종합 배송 플랫폼으로 한 단계 성장합니다.

  • 🎯
    명확한 서비스 구분
    🏍️ 퀵서비스: 오토바이로 작은 물건 / 🚚 화물서비스: 다마스, 라보, 1톤으로 큰 물건
  • 💰
    상하차 서비스별 요금제
    고객이 직접 하면 0원, 기사님이 도와주면 추가 요금 (최대 16,000원)
  • 📱
    간편한 8단계 주문
    물건 크기만 선택하면 적합한 차량이 자동으로 필터링되는 똑똑한 시스템
  • ✨
    직관적인 사용자 경험
    복잡한 설정 없이 아코디언 방식으로 단계별 선택이 쉬워요

더 탄탄한 기술 기반 구축

안정적이고 효율적인 서비스 운영을 위한 기술적 토대를 다집니다

⚙️

운영 효율을 높이는 관리 도구들

사장님들과 고객사가 더 편리하게 서비스를 관리할 수 있도록 지원합니다.

  • 📄
    세금계산서 역발행 원스톱
    기존에 따로 처리하던 세금계산서를 사장님 웹에서 바로 발행할 수 있어요
  • 📢
    고객사별 맞춤 공지 시스템
    고객사가 직접 이미지, 팝업 등으로 원하는 형태의 공지사항을 만들고 관리
  • 💳
    사장님 결제 취소 기능
    삼성E&A 등 대형 제휴점 요청으로 사장님이 직접 결제취소를 처리해서 CS 부담을 줄여요
  • 💰
    부서별 예산 관리
    퀵서비스를 부서 예산 범위 내에서 체계적으로 관리할 수 있어요
